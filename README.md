# Diet Soda | Pure Zero Refreshment 🥤

A modern, highly interactive, and visually stunning landing page for a fictional "Diet Soda" brand. Designed to highlight a premium feel using smooth animations, 3D elements, and modern web design principles.

## ✨ Features

- **Immersive 3D Models:** Powered by `<model-viewer>`, featuring interactive 3D soda cans, floating berries, and dynamic leaves that react to the environment.
- **Advanced Animations:** Smooth scrolling effects, scroll-triggered reveals, and micro-interactions built with **GSAP** (GreenSock Animation Platform).
- **Modern Aesthetics:** Utilizes a sleek dark mode design with glassmorphism (frosted glass) effects, vibrant accents, and high-quality typography.
- **Fully Responsive:** Optimized layout ensuring a flawless experience across all device sizes — from mobile screens to large desktop monitors.
- **Eco-Friendly Theme:** Highlights the brand's commitment to sustainability with a dedicated "Refreshingly Green" section.

## 🚀 Tech Stack

- **HTML5:** Semantic and accessible structure.
- **CSS3:** Custom styling with CSS Variables, Flexbox, Grid, and advanced backdrop filters.
- **JavaScript (ES6):** Logic for interactions and dynamic element manipulation.
- **GSAP (GreenSock):** Core animation engine for scroll and timeline effects.
- **Google Model Viewer:** Web component for rendering interactive 3D models seamlessly.
- **Google Fonts:** Utilizing modern typefaces (Outfit, Inter, Manrope, Galada).
